# BIOS website

Готовая статическая версия сайта BIOS для GitHub Pages.

## Публикация

1. Распакуйте архив.
2. Загрузите содержимое папки `BIOS-GitHub` в корень репозитория GitHub.
3. В настройках репозитория откройте `Settings` → `Pages`.
4. Выберите публикацию из ветки `main`, папка `/ (root)`.

Главная страница: `index.html`.

Дополнительные страницы:

- `patients.html` — для пациентов;
- `clinics.html` — для клиник.

Все необходимые изображения, библиотеки и 3D-данные находятся в папке `assets`.
